Exercício 8

Uma progressão aritmética é uma sequência numérica em que cada termo, a partir do segundo, é igual a soma do termo anterior com uma constante. A constante é calculada como sendo a diferença entre o segundo e o primeiro número. Crie um algoritmo que receba dois números inteiros e, a partir dessa informação, gere uma sequência em progressão aritmética.

Exercicios propostos pelo professor Luiz Paulo Maia - Instituto Infnet/RJ

ATENÇÃO: Rodar pelo console do navegador. Na opção de desenvolvedor do navegador.
